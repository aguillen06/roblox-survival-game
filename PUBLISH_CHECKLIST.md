# Survival Game - Publishing Checklist

## Before Publishing

### Game Settings
- [ ] Game name set
- [ ] Game description written
- [ ] Game icon uploaded (512x512)
- [ ] Thumbnails uploaded (3 recommended)
- [ ] Genre set to "Adventure" or "Action"

### Monetization
- [ ] Private Servers enabled (150 Robux)
- [ ] Create VIP Pass (399 Robux) - update ID in GamePassSystem
- [ ] Create Double Coins pass (149 Robux) - update ID
- [ ] Create Speed Demon pass (99 Robux) - update ID
- [ ] Create Tank pass (149 Robux) - update ID
- [ ] Create Lucky Charm pass (199 Robux) - update ID
- [ ] Create DJ Pass (249 Robux) - update ID

### Sounds (Optional but Recommended)
Upload sound assets and update IDs in `ReplicatedStorage/GameSounds`:
- [ ] LobbyMusic
- [ ] GameMusic
- [ ] FoodPickup
- [ ] PowerUpPickup
- [ ] Damage
- [ ] GameOver
- [ ] ButtonClick
- [ ] Purchase
- [ ] Achievement
- [ ] LevelUp
- [ ] Countdown
- [ ] Victory
- [ ] Defeat
- [ ] Heartbeat
- [ ] Whoosh

### Testing
- [ ] Play through tutorial
- [ ] Test all 4 difficulty levels
- [ ] Test shop purchases
- [ ] Test achievements unlock
- [ ] Test daily rewards
- [ ] Test code redemption
- [ ] Test on mobile device
- [ ] Test leaderboard saving
- [ ] Test private server (if possible)

## After Publishing

### Social Media
- [ ] Create Twitter/X account for game
- [ ] Create Discord server
- [ ] Create YouTube channel (optional)
- [ ] Post launch announcement

### Roblox Promotion
- [ ] Add to groups
- [ ] Submit for sort features
- [ ] Consider Sponsored Ads

### Community
- [ ] Enable in-game chat
- [ ] Monitor for bugs/feedback
- [ ] Plan first update

## First Week Updates

### Quick Wins
- [ ] Fix any reported bugs
- [ ] Add requested features
- [ ] Release new codes for players
- [ ] Thank players on social media

### Content Ideas
- [ ] New map/arena
- [ ] New hazard types
- [ ] New power-ups
- [ ] Seasonal event
- [ ] New achievements
- [ ] Pet system
- [ ] Rebirth system

## Metrics to Track

- Daily Active Users (DAU)
- Average session length
- Retention rates (D1, D7, D30)
- Revenue per user
- Game pass conversion rate
- Code redemption rate
- Achievement completion rate
